# Musicreater 音·创 命令行模式

为了对后期安卓的兼容，先支持命令行

## 命令列表

### exit

- ```/s``` 保存并退出
- ```/c``` 清除日志

### save

- ```/a <文件名>``` 指定文件名

### load

- ```/input <文本>``` 输入音符
- ```/txt <文件名>``` 从文本载入音符
- ```/mid <文件名>``` 解析midi文件载入音符

### win

开启窗口（仅支持的系统下有效）

### chdir <位置>

切换当前工作目录

### build

- ```/file``` 在当前目录生成函数文件（散装）
- ```/directory <文件名>``` 在当前目录生成一个附加包目录
- ```/mcpack <文件名>``` 在当前目录生成一个.mcpack附加包
