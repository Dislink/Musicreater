## 介绍🚀

音·创 Musicreater 是一款免费开源的 **《我的世界：基岩版》** 音乐制作软件

音·创 库版 (Musicreater Package Version) 是一款免费开源的针对 **《我的世界：基岩版》** 的midi音乐转换库

欢迎加群：[861684859](https://jq.qq.com/?_wv=1027&k=hpeRxrYr)
